Эта папка содержит программу и все необходимые файлы, содержащие в том числе и результаты ее работы.

Для запуска необходимы библиотеки: pandas, xlrd, openpyxl

Протестированно: Python 3.8.6

Вход:
 - logs.xlsx - Логи. Входной файл. Не изменяется в процессе работы! Название фиксированно!
 - report.xlsx - Выход. Выходной файл. Изменяется в процессе работы! Название фиксированно!
